# type-speed
Measures typing speed in WPM, and if there are errors, then the WPM is reduced. You can also compare your result with others. This is built as a personal project by Brian Wei.

# Tools Used
  * **React** for front-end
  * **Express.js** for back-end
  * **MongoDB (JSON)** to store users, sentences, results
  * **Google Cloud Build (CI/CD) / Cloud Run** for hosting the website in the cloud
  * **Docker** for installing dependencies, injecting API URL, starting Node server

# Directions
The URL is `https://typing-app-274377411582.us-west2.run.app/`. You can then register an account and log in. Then, choose between "Easy", "Medium", and "Hard". To see the best records, click on "Best Records" which shows the top three users for each difficulty.
